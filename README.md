### 1. Install Dependencies

### 2. Set Up API Key

Get your Gemini API key from: https://makersuite.google.com/app/apikey

Then create a `.env` file:
```bash
echo "GEMINI_API_KEY=your_key_here" > .env
```

Or export directly:
```bash
export GEMINI_API_KEY="your_key_here"
```
